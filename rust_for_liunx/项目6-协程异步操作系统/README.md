# 项目六：基于协程异步机制的操作系统/驱动



### **项目目标：**

基于Rust语言的异步支持和已有的用户态中断、共享调度器和异步驱动框架embassy等工作，在rCore-N或seL4操作系统上引入异步机制，实现内核态或用户态的设备驱动，以及调度器等内核模块的异步改造。

### **项目内容：**

1. Rust语言的异步机制；

2. 用户态中断；

3. 设备驱动的异步框架embassy；

4. 以协程为调度单位的共享调度器；
